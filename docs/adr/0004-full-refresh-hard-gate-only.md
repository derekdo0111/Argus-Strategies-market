# ADR-0004: 全量刷新只跑到硬门

- **日期**: 2026-06-15
- **状态**: 已接受

## 上下文

龟龟策略的四个门中，现金质量门和穿透回报率门是硬门（确定性规则），基本面门和估值门是软门（需要 LLM 分析）。

全量刷新时，需要考虑 LLM 调用成本和速度。

## 决策

全量刷新（定时任务）只跑到硬门：
1. 选股器筛选候选池
2. 拉取全量 Tushare 数据缓存
3. 现金质量门判定（确定性计算）
4. 穿透回报率门判定（确定性计算）
5. 输出股池列表（按 PR 降序）

基本面门和估值门在用户点击单股时按需触发（LLM 分析）。

## 后果

- 全量刷新速度快，无需 LLM 调用
- 股池数据可每日更新
- 单股分析首次点击时有延迟（需等待 LLM 分析完成）
- 分析结果可缓存，二次点击即时返回
