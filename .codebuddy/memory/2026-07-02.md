## v0.18.0 股池筛选：业务纯度排名

### 背景
Brainstorming session 发现三个结构缺陷：
- 业务纯度盲区：永鼎股份超导仅占营收16%，但因光纤AI增速+卖房收益排第一
- 一次性损益污染：利润增速被卖房等非经常性收益污染
- 动量绝对值分档：与其他因子的行业内百分位排名不一致

### 设计决策（逐项确认）
1. **纯度分（C方案-重量级）**：新增第三维度 → 最终简化为纯度分唯一排名依据
2. **数据源**：Tushare `fina_mainbz_vip`（分业务收入，结构化数据）
3. **映射方式**：线性映射（占比 = 纯度分，16% → 0.16）
4. **角色分配**：方向分仅标注 matched_l3、财务分仅参考（原始指标展示）
5. **最终公式**：排名 = 纯度分，报告表格：排名|股票|纯度分|ROE|毛利率|营收增速|匹配方向

### 实现
- 新增: `purity_scorer.py`（批量拉取 → LLM匹配业务线 → 计算占比）
- 新增: `tushare_client.py.get_fina_mainbz()`
- 修改: `stock_screener.py`（去动量，权重营收35/利润35/ROE20/质量10）
- 修改: `screening_agent.py`（4阶段：方向标注→纯度打分→财务参考→纯度排名）
- 修改: `report_agent.py`（表格列更新）
- 清理: `config.py`删除`PROSPERITY_DIRECTION_WEIGHT`/`PROSPERITY_FINANCE_WEIGHT`
- 清理: `.env`删除旧权重配置
- 测试: 更新+新增7个测试，41/41 全通过

### 预期效果
可控核聚变：永鼎股份 1→4+（纯度0.16），西部超导 3→1（纯度~0.55）
