- 现在**任意** `[文字](#锚点)` 格式的 markdown 内部链接都会被加前缀

### 测试

## v0.7.11 — 股池 QRV 评分栏不显示分数 (根本修复)

### 背景

- `qrv_agent.py` 新增 `_parse_scores()` 静态方法 (~50行)：用正则从 LLM markdown 输出的综合打分卡表格提取 10 维度分数 + 综合总分 + 自动算 Q/R/V 加权均分
- `analyze_async()` 写 JSON 前调 `_parse_scores()`，scores 随 JSON 写入
- 已有的 `stocks.py` 回填逻辑现在终于能拿到 scores 数据，前端即时显示

### 向后兼容
- 已分析个股需**重新分析**才能在 JSON 中生成 scores（不重新分析 → scores=None → `"—"`，无破坏）
- `_parse_scores()` 对 N/A 行自动跳过，无表格时返回 None
