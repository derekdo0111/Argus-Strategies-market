## v0.6.2 前端白屏+加载慢修复 (23:46)
- **问题**: 页面白屏一片 + 加载很慢
- **根因分析**:
  1. CSS 高度塌陷 — `html`/`body` 无 `height: 100%`，`#root { height: 100vh }` 在无完整高度链条时布局缩为 0 → 白屏
  2. Pool API 每次请求遍历 43 个目录 × 读 2 个 YAML = 86 次 `yaml.safe_load()`，极慢
  3. StockPool 无 `isError` 处理，API 出错只显示「暂无数据」
- **修复**:
  1. `frontend/src/index.css`: `html` 加 `height: 100%`，`body` 加 `height: 100%` + `min-height: 100vh`，`#root` 从 `100vh` 改为 `100%` + `min-height: 100vh`
  2. `backend/app/api/stocks.py`: `get_stock_pool` 加内存缓存 `_pool_cache` (TTL=300s)，命中直接分页返回
  3. `frontend/src/components/StockPool.tsx`: 解构 `isError`/`error`/`refetch`，新增错误状态 UI（错误消息 + 重试按钮）
  4. `frontend/src/components/StockPool.module.css`: 新增 `.error`/`.retryBtn` 样式
  5. `CHANGELOG.md`: v0.6.2 条目
- **零 lint 错误**
- **待执行**: 重启后端 `uvicorn` 使缓存生效 + 启动前端 `npm run dev`
